Document Intelligence Platform Test

Introduction

This is a test HTML document to verify the document ingestion pipeline's HTML parsing capabilities.

Features

  • HTML tag stripping
  • Text extraction
  • Structure preservation
  • Metadata handling

Technical Details

The system uses BeautifulSoup to parse HTML and extract clean text content while removing scripts, styles, and other non-content elements.

Example Code

const analyzeDocument = async (docId) => {
    const result = await fetch(`/api/v1/analyze`, {
        method: 'POST',
        body: JSON.stringify({ document_ids: [docId] })
    });
    return result.json();
};
